# Bank Account Simulation

**EN** — Console Java app (Java 21) simulating customers and bank accounts (standard, checking, savings). Practice project: OOP, inheritance, transactions, console commands. Run main class `BankAccountSimulation` from `bankAccountSimulation/`.

**HU** — Konzolos Java alkalmazás (Java 21): ügyfelek és bankszámlák (normál, folyószámla, megtakarítás) szimulálása. Gyakorlóprojekt: OOP, öröklődés, tranzakciók, parancssor. Futtatás: `BankAccountSimulation` main a `bankAccountSimulation/` mappából.
